== Description ==

Quick plugin to disable all default WordPress canonical links on your pages. Can be used if you're generating canonical links
through other means, to avoid duplicate link entries.

For more information on canonical links, see [Matt Cutt's blog entry](http://www.mattcutts.com/blog/canonical-link-tag/)

== Installation ==

* Install the widget using Add New from the Plugins menu in the WordPress admin dashboard. Search for Remove Canonical Links.
* Once activated, this plugin simply unregisters WordPress' built in canonical link generation, removing them from all pages.
* Implement canonical links through your own preferred method of third party plugins or themes.
